# ⚡ Démarrage Rapide - Interface Admin

## Installation

```bash
cd admin
npm install
```

## Configuration

Modifier `src/environments/environment.ts` :

```typescript
apiUrl: 'http://localhost:8080',
```

## Lancer

```bash
npm start
```

L'interface sera accessible sur : **http://localhost:4200**

## Connexion

1. Entrer votre email admin
2. Cliquer sur "Envoyer OTP"
3. Vérifier la console backend pour l'OTP
4. Entrer l'OTP et se connecter
